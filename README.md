# LethalCompanyRadar
A PowerShell radar script for Lethal Company
1. YOU NEED TO HAVE BEPINEX FROM THUNDERSTORE (MAYBE IT WORKS ON OTHER CLIENTS)
2. OPEN THE SCRIPT BY LEFT CLICKING IT AND PRESSING RUN WITH POWERSHELL
3. FIND THE PATH OF THE LOGS
(OPEN THUNDERSTORE, OPEN THE PROFILE YOU PLAY ON WITH BEPINEXPACK INSTALLED, GO TO SETTINGS → BROWSE DATA FOLDER → LETHALCOMPANY/PROFILES/...PROFILE YOU USE.../BEPINEX/LOGOUTPUT.LOG, DRAG LOGOUTPUT.LOG TO THE SCRIPT AND PRESS ENTER)
4. YOU’RE DONE! MAKE SURE SYSTEM VOLUME IS UP AND IT SHOULD WORK
